# Automated management of Network Systems

Supervisor: Eduardas Kutka

System that continuously monitors a network, detects anomalies or faults in real-time, and automatically triggers remediation processes to restore or optimize network performance. The project combines elements of network monitoring, anomaly detection, and automated remediation to achieve a self-healing infrastructure.

### Tools:

- Ansible
- Grafana
- Prometheus
- Nginx
- Python
